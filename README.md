# Mi Barrio en Cifras

> Neighborhood demographics and public spending visualizer — Colombia open data.

Interactive 3D scrollytelling map showing neighborhood-level demographics, public investment, and quality of life indicators across Colombian cities. Built with Chart.js and MapLibre GL JS.

**Stack:** Vanilla JS · MapLibre GL JS · Chart.js · HTML · CSS

## Run locally

```bash
npx serve .
# or
python3 -m http.server 8080
```

Open `http://localhost:8080`

## Data sources

- DANE — Departamento Administrativo Nacional de Estadística
- SECOP II — Public contracting data
